# Milk Trace System

## Tech Stack

- Vite + Vitest
- React + Recoil
- TypeScript
- Sass
- Semi-Design
- ESLint + StyleLint + EditorConfig
- CommitLint + Husky + Lint-Staged
- Standard-Version
- Github workflows

```bash
pnpm i
```
